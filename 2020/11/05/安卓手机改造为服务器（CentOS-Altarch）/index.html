安卓手机改造为服务器（CentOS-Altarch）

发表于 2020-11-05 阅读次数：
在旧的安卓手机上安装 Linux Deploy 以改造成服务器

第一步：获取手机 ROOT 权限#

不同手机方式不同，自行查找

下载apk#

Busybox

Linux Deploy

Juicessh

安装 Busybox#

正常安装即可

安装 Linux Deploy#

安装完Linux deploy之后：
首先点开左侧菜单，然后点击“设置”，【锁定wifi】，【CPU唤醒】勾上，【PATH变量】最重要，点击并填上/system/xbin，这样才能关联Busybox。然后点击下面的【更新环境】。

然后点击右下角的设置图标开始进行配置，这里以Centos为例：
发行版：【CentOS】
架构：【aarch64】
版本：【7】
源地址：http://mirrors.aliyun.com/centos-altarch/

镜像大小，可以不改（似乎默认是2G 最好改大点

点击编辑用户名和用户密码分别进行设置。

勾选 启用 ssh 服务器

点击右上角三个点 选择安装

等待约10分钟 安装完成

安装 Juicessh#

基本设置中 昵称自选，地址填写127.0.0.1（回环地址）

选择认证 –> 新建 –> 用户名填服务器用户名，密码填服务器密码 完成后点击右上角 √ 进行确认完成

再点击右上角 √ 进行确认完成

在点击连接即可

设置系统编码#

安装nodejs#

在以下网址找到合适的版本 https://npm.taobao.org/mirrors/node/

这里使用 https://npm.taobao.org/mirrors/node/v15.1.0/node-v15.1.0-linux-arm64.tar.gz

下载完成后

修改Centos-Altarch镜像#

Hexo 搭建博客#

按 hexo官网 进行操作

完成后 在同个局域网内 访问 [ip]:[port] 即可 port默认是4000
