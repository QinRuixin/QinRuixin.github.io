Dockerfile简介与基本命令

Dockerfile 是用来构建 Docker 镜像的构建文件，是由一系列命令和参数构成的脚本

  • 编写 Dockerfile 文件
  • docker build
  • docker run

镜像是分层的， Dockerfile 的每一条命令都会在之前镜像的基础上添加一层

基础知识#

  1. 每条保留字指令 大写 且后面要跟随至少一个参数
  2. 指令按照从上到下，顺序执行
  3. #表示注释
  4. 每条指令创建一个新的镜像层，并对镜像进行提交
